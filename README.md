\# Sales Data Simulation and Forecasting Using NumPy



\## Project Overview

This project simulates and forecasts product sales data using NumPy.



The project demonstrates how numerical computing and statistical analysis can be applied to simulate real-world business scenarios such as sales forecasting and performance analysis.



\## Features

\- Sales data simulation

\- Random market variation modeling

\- Monthly sales forecasting

\- Statistical analysis

\- Data visualization using Matplotlib



\## Skills Demonstrated

\- NumPy array operations

\- Statistical analysis

\- Forecasting techniques

\- Data simulation

\- Data visualization



\## Technologies Used

\- Python

\- NumPy

\- Matplotlib

\- Jupyter Notebook



\## Key Insights

\- Identified best-performing products

\- Calculated average and total sales

\- Forecasted future sales growth

\- Analyzed variation in product performance
